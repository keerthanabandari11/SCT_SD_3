# Sudoku Solver

## Description
This project automatically solves Sudoku puzzles using the Backtracking Algorithm.

## Features
- Solves incomplete Sudoku puzzles
- Uses recursion and backtracking
- Displays solved Sudoku board

## Technologies Used
- Python

## Algorithm Used
- Backtracking Algorithm

## How to Run

```bash
python sudoku_solver.py
Example Input

[5,3,0,0,7,0,0,0,0]
[6,0,0,1,9,5,0,0,0]

Example Output

Solved Sudoku Puzzle
